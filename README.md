# GearHub MKT

Plataforma digital voltada à compra e venda de peças automotivas, desenvolvida para atender o público entusiasta (gearheads) com uma experiência moderna, rápida e intuitiva.

O projeto tem como foco conectar compradores e vendedores em um ambiente otimizado, com ênfase em usabilidade, performance e conversão.

---

## 📌 Visão Geral

O **GearHub MKT** é uma aplicação web que centraliza a negociação de peças automotivas, oferecendo uma interface simplificada para descoberta, análise e tomada de decisão.

A plataforma foi pensada para atender desde usuários iniciantes até entusiastas mais experientes, proporcionando uma navegação fluida e objetiva.

---

## 🎯 Objetivos do Projeto

- Facilitar a compra e venda de peças automotivas  
- Atender um nicho específico: entusiastas automotivos (gearheads)  
- Criar uma experiência de navegação rápida e intuitiva  
- Aplicar boas práticas de UX/UI com foco em conversão  
- Construir uma base escalável para evolução da plataforma  

---

## 🚗 Público-Alvo

- Entusiastas automotivos (gearheads)

## 🚀 Como Executar o Projeto

### Pré-requisitos

- Node.js (versão 18 ou superior)  
- npm ou yarn  

### Instalação

```bash
git clone https://github.com/RenanSchmitt/gearhubmkt.git
cd gearhubmkt
npm install
Execução
npm run dev
Build para Produção
npm run build
```
